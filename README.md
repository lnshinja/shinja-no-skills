# shinja-no-skills

LN教（Lightning Network Church）関連のOpenClawスキルをまとめるリポジトリです。

## Contents
- `lnchurch-site/` — LN教公式サイト（kari.mayim-mayim.com）の仕様・知識ベースを参照するスキル

## Usage
1. このリポジトリから使いたいスキルを取得
2. OpenClawのスキル配置先にスキルフォルダを置く（例: `skills/`）
3. `.skill` が必要な場合は `package_skill.py` でパッケージ

> 例: `package_skill.py path/to/skill`

## Notes
- スキルの詳細は各ディレクトリ内の `SKILL.md` を参照してください
- 公式仕様の更新に合わせて内容を随時更新します
